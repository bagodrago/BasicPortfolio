# Basic Portfolio Website
This website project is meant as a tool to share my previous projects, what I learned from them, and provide links to the Github repositories for those interested.

This project is a means for me to learn website design through Next.js using tools like TailwindCSS, DaisyUI, and Lucide to make a simple, clean looking website. I intend to continue developing these skills in other projects. This will be forked into bagodrago.github.io inevitably when I host it.
